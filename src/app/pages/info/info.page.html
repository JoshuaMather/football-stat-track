Info This app keeps track of players and matches with accompanying statistics. Players and their details can be added through the players list.
Once created a player will be visible in the list and be able to be added to matches.
The player details can be viewed in the list and they can be edited.
Players statistics will be tracked, which can be viewed through the player's entry in the player list.
These statistics are updated per match, where statistics for the players that played are set.
Any subs owed can also be tracked via the player list. Matches that are played can be added via the matches list.
When adding a match, the match info, teams, and statistics for each player for the match can be added.
As matches can be edited, not all these need be set initially.
Once a match is created, it can be seen in the list, where it can be edited or have its details viewed.
From the matches list, total stats and leaderboards can be viewed.
